# Pay Slip Genie

## Project info

This is a Pay Slip Generator application built with React, TypeScript, and Supabase.

## Getting Started

To get started with this project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
